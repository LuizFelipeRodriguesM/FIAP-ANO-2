# Cyphin — FIAP PBL 2026

Repositorio de entregas do PBL (Problem Based Learning) da FIAP — Engenharia de Software, 2o ano (EAD), 2026.

## Projeto

**Cyphin** — copiloto financeiro com IA para investidores de criptoativos.

Cenario ficticio: a empresa **VOLTZ** recebeu um pedido do **Tio Patinhas** (PO) para construir uma plataforma web de investimentos em cripto. Cada fase do ano entrega um artefato diferente do mesmo projeto.

## Como navegar o repo

Cada fase fica em `fase-N/` e sempre tem:

- `README.md` — **comece por aqui** — explica o que e cada arquivo da pasta
- `enunciado.md` — o que a FIAP pediu
- `entrega.md` (quando aplicavel) — documento oficial entregue

Alem desses, a fase pode ter arquivos de apoio (diagramas, roteiros, explicacoes, TODO) — o `README.md` de cada fase lista todos.

## Como contribuir

1. Abrir `fase-N/README.md` pra entender o que ja existe
2. Ler `enunciado.md` pra saber o que a FIAP pediu
3. Trabalhar nos artefatos da fase
4. Atualizar o `README.md` da fase se adicionar arquivos novos

## Contexto do PO (Tio Patinhas)

Requisitos que orientam todas as fases:

1. Investir em criptoativos de maneira online
2. Acompanhamento diario dos investimentos
3. Seguranca dos dados e valores trafegados em rede
4. Possui varias empresas de onde envia valores para investimentos
5. Acesso exclusivamente via sistema WEB (nao confia em app)
